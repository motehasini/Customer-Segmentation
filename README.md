# Project Overview

This project performs customer segmentation using the K-Means Clustering Machine Learning algorithm.

The objective of this project is to group mall customers into different clusters based on their:

- Annual Income
- Spending Score

Customer segmentation helps businesses understand customer behavior and improve marketing strategies.

# Machine Learning Technique Used

## K-Means Clustering

K-Means Clustering is an unsupervised Machine Learning algorithm used to group similar data points into clusters.

In this project:

- Customers were segmented based on income and spending behavior.
- The Elbow Method was used to determine the optimal number of clusters.
